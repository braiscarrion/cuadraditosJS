# cuadraditosJS

Just a classic paper game made in JS

## Requirements

- [NodeJS & NPM](http://nodejs.org/download)

## Installation

Run from your terminal:

    git git@github.com:braiscarrion/cuadraditosJS.git
    cd cuadraditosJS\bin\server
    npm install

## Run

    npm start
